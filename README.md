# CS442-Cooking-Recipe-Database-Project
This is our semester project for Database Systems (CS 44200).

This project contains:
- **Backend**: Spring Boot
- **Frontend**: Next.js + React + TypeScript
- **Database**: PostgreSQL

---

## 1. Prerequisites

Install these first:
- **Git**
- **Java 17+ JDK**
- **Node.js 20+**
- **PostgreSQL 15+**

## 2. Clone the repository

Clone the repository onto your machine

## 3. Set Up the Database

1. **Start PostgreSQL** on your machine.
2. Log into PostgreSQL from the terminal
3. In the psql prompt run the **create.sql** and **populate.sql** scripts in order
